## ✅ 使用前检查清单

在开始配置之前,请确认:

- [ ] 手机是Android系统(iOS不支持)
- [ ] Android版本7.0或更高
- [ ] 已安装企业微信
- [ ] 已准备好群聊邀请链接
- [ ] 手机电量充足(建议>50%)
- [ ] 网络连接正常

---

## 📱 获取群聊邀请链接

### 方法1: 企业微信后台

```
1. 登录企业微信管理后台
2. 客户联系 → 客户群
3. 选择群聊 → 生成邀请链接
4. 复制链接
```

### 方法2: 手机端

```
1. 打开企业微信
2. 进入群聊
3. 点击右上角"..."
4. 群二维码 → 分享链接
5. 复制链接
```

## ⚠️ 重要提醒

### 权限要求

使用本方案需要授予以下权限:
- **无障碍权限** - 必须,用于自动操作
- **通知访问权限** - 必须,用于监听通知
- **悬浮窗权限** - 建议,用于显示状态

### 注意事项

- 🚨 请勿频繁操作,避免被企业微信检测
- 🚨 建议设置合理的延迟时间(2-5秒)
- 🚨 定期检查自动化规则是否正常运行
- 🚨 企业微信更新后可能需要重新配置

### 风险提示

- 本方案使用无障碍服务模拟人工操作
- 虽然风险较低,但仍建议合理使用
- 避免短时间大量自动回复
- 建议设置工作时间限制
