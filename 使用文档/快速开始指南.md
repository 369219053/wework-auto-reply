# 🚀 5分钟快速开始指南

## ✅ 前提条件检查

在开始之前,请确认:

- [ ] Android手机(Android 7.0或更高版本)
- [ ] 已安装企业微信
- [ ] 已准备好群聊邀请链接
- [ ] 手机电量充足(建议>50%)
- [ ] 网络连接正常

---

## 📋 第一步: 安装MacroDroid (2分钟)

### 方法1: Google Play安装(推荐)
1. 打开Google Play商店
2. 搜索"MacroDroid"
3. 点击"安装"
4. 等待安装完成

### 方法2: APK安装
1. 访问MacroDroid官网下载APK
2. 允许安装未知来源应用
3. 点击APK文件安装

### 安装后首次打开
1. 打开MacroDroid应用
2. 点击"开始使用"
3. 跳过教程(或快速浏览)

---

## 📋 第二步: 授予必要权限 (2分钟)

### 1. 无障碍权限(必须)

**操作步骤:**
```
打开MacroDroid 
→ 点击右上角"三条横线"菜单
→ 选择"设置"
→ 点击"配置无障碍服务"
→ 找到"MacroDroid"并开启
→ 点击"允许"
```

**为什么需要:**
- 用于自动点击通知
- 用于自动输入文本
- 用于自动发送消息

### 2. 通知访问权限(必须)

**操作步骤:**
```
打开MacroDroid
→ 设置
→ 点击"通知访问权限"
→ 找到"MacroDroid"并开启
```

**为什么需要:**
- 用于监听企业微信的新好友通知

### 3. 悬浮窗权限(建议)

**操作步骤:**
```
打开MacroDroid
→ 设置
→ 点击"悬浮窗权限"
→ 允许显示悬浮窗
```

**为什么需要:**
- 用于显示自动化执行状态

---

## 📋 第三步: 创建自动化规则 (5分钟)

### 1. 创建新宏

```
打开MacroDroid
→ 点击右下角"+"按钮
→ 输入名称: "企微自动回复"
→ 点击"确定"
```

### 2. 添加触发器(Trigger)

```
点击"Triggers"区域的"+"
→ 选择"Notification"(通知)
→ 选择"企业微信"应用
→ 标题过滤: 输入"新的朋友"
→ 点击"确定"
```

**提示:** 如果找不到企业微信,请先打开企业微信,然后返回MacroDroid重新选择。

### 3. 添加动作序列(Actions)

#### 动作1: 等待2秒
```
点击"Actions"区域的"+"
→ 选择"Wait"(等待)
→ 输入"2"秒
→ 点击"确定"
```

#### 动作2: 点击通知
```
点击"Actions"区域的"+"
→ 选择"Click Notification"(点击通知)
→ 选择"企业微信"
→ 点击"确定"
```

#### 动作3: 等待1秒
```
点击"Actions"区域的"+"
→ 选择"Wait"(等待)
→ 输入"1"秒
→ 点击"确定"
```

#### 动作4: 输入文本
```
点击"Actions"区域的"+"
→ 选择"Type Text"(输入文本)
→ 输入欢迎语:
   "您好!欢迎添加,请点击链接加入我们的群聊: [您的群聊链接]"
→ 点击"确定"
```

**重要:** 请将`[您的群聊链接]`替换为实际的群聊邀请链接!

#### 动作5: 发送消息
```
点击"Actions"区域的"+"
→ 选择"Press Key"(按键)
→ 选择"Enter"
→ 点击"确定"
```

#### 动作6: 返回主屏幕
```
点击"Actions"区域的"+"
→ 选择"Go Home"(返回主屏幕)
→ 点击"确定"
```

### 4. 保存规则

```
点击右上角"✓"保存
→ 规则自动启用
```

## 🔧 常见问题快速解决

### 问题1: 没有自动执行

**解决方法:**
1. 检查MacroDroid是否有无障碍权限
2. 检查规则是否已启用(绿色开关)
3. 检查通知过滤条件是否正确

### 问题2: 发送的消息不对

**解决方法:**
1. 编辑规则中的"Type Text"动作
2. 修改欢迎语内容
3. 保存并重新测试

### 问题3: 自动化执行太慢

**解决方法:**
1. 减少"Wait"动作的等待时间
2. 建议最少保持1秒延迟,避免出错
