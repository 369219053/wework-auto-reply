## 🔧 安装和配置问题

### Q1: 在哪里下载MacroDroid?

**A:** 有两种方式:
1. **Google Play**(推荐): 搜索"MacroDroid"直接安装
2. **官网下载**: 访问 https://www.macrodroid.com/ 下载APK

### Q2: 免费版够用吗?

**A:** 完全够用!
- 免费版支持最多5个宏
- 企微自动回复只需要1个宏
- 基础功能完全免费

### Q3: 需要root权限吗?

**A:** 不需要!
- MacroDroid不需要root权限
- 只需要授予无障碍权限即可

### Q4: 如何导入配置文件?

**A:** 
```
1. 下载.macro配置文件
2. 打开MacroDroid
3. 点击右上角"三条横线"菜单
4. 选择"导入"
5. 选择下载的.macro文件
```

## 🔐 权限问题

### Q5: 为什么需要无障碍权限?

**A:** 无障碍权限用于:
- 自动点击通知
- 自动输入文本
- 自动点击发送按钮
- 自动返回主屏幕

**这是实现自动化的核心权限!**

### Q6: 如何授予无障碍权限?

**A:** 
```
方法1(推荐):
MacroDroid → 设置 → 配置无障碍服务
→ 找到MacroDroid → 开启

方法2:
手机设置 → 辅助功能 → 无障碍
→ 找到MacroDroid → 开启
```

### Q7: 无障碍权限总是被自动关闭怎么办?

**A:** 这是部分手机厂商的限制,解决方法:

**小米手机:**
```
1. 设置 → 应用设置 → 应用管理
2. 找到MacroDroid
3. 自启动 → 开启
4. 省电策略 → 无限制
5. 锁定后台(在最近任务中下拉锁定)
```

**华为手机:**
```
1. 设置 → 应用 → 应用启动管理
2. 找到MacroDroid → 手动管理
3. 允许自启动、允许关联启动、允许后台活动
4. 设置 → 电池 → 应用启动管理
5. MacroDroid → 手动管理 → 全部允许
```

**OPPO/Vivo手机:**
```
1. 设置 → 电池 → 应用耗电管理
2. MacroDroid → 允许后台运行
3. 设置 → 应用管理 → 权限管理
4. 自启动 → MacroDroid → 允许
```

### Q8: 通知访问权限在哪里授予?

**A:** 
```
MacroDroid → 设置 → 通知访问权限
→ 找到MacroDroid → 开启
```

## ⚙️ 执行问题

### Q9: 为什么没有自动执行?

**A:** 请按以下步骤排查:

**步骤1: 检查权限**
```
- 无障碍权限是否开启?
- 通知访问权限是否开启?
```

**步骤2: 检查宏状态**
```
- 宏是否已启用?(绿色开关)
- 触发器是否配置正确?
```

**步骤3: 检查后台运行**
```
- MacroDroid是否被系统清理?
- 是否加入自启动白名单?
- 是否关闭电池优化?
```

**步骤4: 检查通知过滤**
```
- 通知标题是否包含"新的朋友"?
- 企业微信通知是否正常显示?
```

### Q10: 如何查看执行日志?

**A:** 
```
打开MacroDroid
→ 点击右上角"三条横线"
→ 选择"日志"
→ 查看最近的执行记录
```

### Q11: 执行了但没有发送消息?

**A:** 可能原因和解决方法:

**原因1: 等待时间太短**
```
解决: 增加Wait动作的等待时间
- 点击通知前等待2-3秒
- 输入文本前等待1-2秒
```

**原因2: 输入框未找到**
```
解决: 检查Type Text动作
- 确认文本内容正确
- 尝试增加等待时间
```

**原因3: 发送按钮未点击**
```
解决: 检查Press Key动作
- 确认选择的是Enter键
- 尝试用"Click UI Element"代替
```

### Q12: 发送的消息不完整?

**A:** 
```
原因: 文本太长或输入速度太快
解决:
1. 缩短欢迎语内容
2. 在Type Text后增加Wait 0.5秒
3. 分多次输入文本
```

## 🎯 功能问题

### Q13: 如何修改欢迎语内容?

**A:** 
```
1. 打开MacroDroid
2. 点击"企微自动回复"宏
3. 找到"Type Text"动作
4. 点击编辑
5. 修改文本内容
6. 保存
```

### Q14: 如何获取群聊邀请链接?

**A:** 

**方法1: 企业微信后台**
```
1. 登录企业微信管理后台
2. 客户联系 → 客户群
3. 选择群聊 → 生成邀请链接
4. 复制链接
```

**方法2: 手机端**
```
1. 打开企业微信
2. 进入群聊
3. 点击右上角"..."
4. 群二维码 → 分享链接
5. 复制链接
```

### Q15: 可以发送图片或小程序吗?

**A:** 
- **MacroDroid**: 只能发送文本,不支持图片和小程序
- **Hamibot**: 可以通过脚本实现点击图片/小程序发送
- **定制APK**: 可以完全自定义发送内容

### Q16: 可以根据不同时间发送不同的欢迎语吗?

**A:** 可以!使用If/Then/Else动作:

```
Actions → + → If/Then/Else
- Condition: Time
  - If 09:00-12:00: Type "早上好!欢迎添加..."
  - Else If 12:00-18:00: Type "下午好!欢迎添加..."
  - Else: Type "晚上好!欢迎添加..."
```

### Q17: 可以只在工作时间自动回复吗?

**A:** 可以!使用Constraints约束:

```
Constraints → + → Time
- Start Time: 09:00
- End Time: 18:00

Constraints → + → Day of Week
- 选择: 周一到周五
```

### Q18: 如何记录每次自动回复?

**A:** 添加日志记录动作:

```
Actions → + → Write to Log
- Text: "已自动回复: {notification_title}"
- Log File: /sdcard/企微自动回复日志.txt
```

## 📱 手机兼容性问题

### Q19: 支持哪些Android版本?

**A:** 
- **最低要求**: Android 7.0 (Nougat)
- **推荐版本**: Android 9.0+ (Pie及以上)
- **最佳体验**: Android 11.0+

### Q20: iOS可以使用吗?

**A:** 
- **不可以!** MacroDroid只支持Android
- iOS系统限制太多,无法实现真正的自动化
- iOS用户建议使用企业微信官方API方案

### Q21: 哪些手机品牌兼容性最好?

**A:** 

**兼容性排名:**
1. ⭐⭐⭐⭐⭐ **原生Android** (Google Pixel等)
2. ⭐⭐⭐⭐ **一加、魅族** (系统限制较少)
3. ⭐⭐⭐ **小米、华为** (需要额外配置)
4. ⭐⭐ **OPPO、Vivo** (后台限制较多)

**注意:** 所有品牌都可以使用,只是配置难度不同!

### Q22: 鸿蒙系统可以使用吗?

**A:** 
- **鸿蒙2.0**: 可以,基于Android兼容
- **鸿蒙3.0+**: 可以,但需要额外配置权限
- **纯鸿蒙**: 未来可能不支持

**A:**

## 🆘 其他问题

### Q26: 会被企业微信检测封号吗?

**A:** 
- **风险较低**: MacroDroid只是模拟人工操作
- **建议**: 设置合理的延迟时间(2-5秒)
- **避免**: 频繁操作,短时间大量回复

### Q27: 耗电量大吗?

**A:** 
- **正常情况**: 耗电很少,几乎可以忽略
- **后台运行**: 不会显著增加耗电
- **建议**: 关闭不必要的日志记录

### Q28: 需要一直开着屏幕吗?

**A:** 
- **不需要!** 可以锁屏运行
- MacroDroid在后台持续监听
- 收到通知时自动唤醒执行

### Q29: 可以同时监听多个企业微信账号吗?

**A:** 
- **不可以**: 手机只能登录一个企业微信
- **替代方案**: 使用企业微信官方API方案
