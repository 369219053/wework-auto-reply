📖 项目简介

企业微信自动化应用是一款基于Android无障碍服务开发的自动化工具,帮助企业高效管理客户关系。

应用特点

  • ✅ 纯手机端运行 - 无需电脑,手机独立完成所有操作
  • ✅ 一键执行 - 点击按钮即可完成批量操作
  • ✅ 智能识别 - 自动识别UI元素,适配不同版本
  • ✅ 安全可靠 - 本地运行,不上传任何数据
---

✨ 核心功能

功能一: 批量通过好友申请并邀请进群

适用场景: 企业客服、社群运营、私域流量管理

功能流程: 


1. 自动批量通过所有好友申请
2. 记录所有新添加的客户名称
3. 自动将所有新客户邀请到指定群聊
4. 显示操作结果统计

操作步骤: 1. 打开应用,点击"功能一"标签 2. 输入目标群聊名称(例如: 智界Aigc客户群) 3. 点击"开始批量处理"按钮 4. 应用自动执行,完成后显示统计结果

预期结果:

  • ✅ 所有好友申请已通过
  • ✅ 所有新客户已加入群聊
  • ✅ 显示成功/失败统计
---

功能二: 批量发送消息

适用场景: 群发通知、营销推广、客户维护

功能流程: 


1. 从素材库聊天选择要转发的消息
2. 自动转发到多个群聊/联系人
3. 支持文本、图片、卡片等多种消息类型
4. 智能滚动查找不可见的联系人

操作步骤: 1. 打开应用,点击"功能二"标签 2. 点击"设置"配置素材库聊天名称 3. 创建消息组,添加目标群聊/联系人 4. 设置转发消息数量和延迟时间 5. 点击"开始发送"按钮 6. 应用自动执行,完成后显示统计结果

预期结果:

  • ✅ 消息已发送到所有目标群聊
  • ✅ 显示成功/失败统计

📦 安装步骤

步骤1: 获取APK文件

方法A: 从开发者获取

  • 联系开发者获取最新版APK文件
  • 通过微信/QQ接收APK文件
方法B: 从GitHub下载
  • 访问项目GitHub仓库
  • 进入Releases页面
  • 下载最新版APK文件

步骤2: 安装APK

1. 允许安装未知来源应用 - 打开手机"设置" - 搜索"安装未知应用" - 找到"文件管理器"或"浏览器" - 开启"允许安装未知应用"

2. 安装APK文件 - 找到下载的APK文件 - 点击APK文件 - 点击"安装"按钮 - 等待安装完成

3. 打开应用 - 安装完成后点击"打开" - 或在桌面找到"企微自动回复"图标

步骤3: 授予必要权限

无障碍服务权限(必须)

这是应用的核心权限,用于自动操作企业微信。

开启方法: 


方法1(推荐):
应用内提示 → 点击"去设置" → 找到"企微自动回复" → 开启
方法2:
手机设置 → 无障碍 → 已下载的应用 → 企微自动回复 → 开启

其他权限:

  • ✅ 存储权限 - 用于保存数据
  • ✅ 通知权限 - 用于显示操作进度
---

🔧 权限配置

不同品牌手机需要额外配置,确保应用能在后台正常运行。

小米/红米 (MIUI)

1. 关闭MIUI优化 


设置 → 更多设置 → 开发者选项 → 关闭"MIUI优化"

2. 允许后台运行 


设置 → 应用设置 → 应用管理 → 企微自动回复
→ 省电策略 → 无限制

3. 自启动权限 


设置 → 应用设置 → 应用管理 → 企微自动回复
→ 自启动 → 开启

华为/荣耀 (EMUI/HarmonyOS)

1. 加入受保护应用 


设置 → 电池 → 应用启动管理 → 企微自动回复
→ 手动管理 → 全部允许

2. 关闭电池优化 


设置 → 电池 → 更多电池设置 → 休眠时始终保持网络连接

OPPO/Vivo (ColorOS/OriginOS)

1. 关闭电池优化 


设置 → 电池 → 更多 → 耗电保护 → 企微自动回复 → 关闭

2. 允许后台运行 


设置 → 应用管理 → 企微自动回复 → 电池 → 后台冻结 → 关闭

三星 (OneUI)

1. 关闭电池优化 


设置 → 应用程序 → 企微自动回复 → 电池 → 优化电池使用 → 关闭

📱 使用方法

功能一: 批量通过好友申请并邀请进群

准备工作

1. 确保有待处理的好友申请 - 打开企业微信 - 点击"通讯录"标签 - 查看"新的客户"是否有待处理申请

2. 准备目标群聊 - 确认群聊名称(例如: 智界Aigc客户群) - 确保您是群主或有邀请权限

操作步骤

第1步: 打开应用

  • 在桌面找到"企微自动回复"图标
  • 点击打开应用
第2步: 选择功能一
  • 点击底部"功能一"标签
  • 进入批量处理页面
第3步: 输入群聊名称
  • 在输入框中输入目标群聊名称
  • 例如: 智界Aigc客户群
  • 注意: 只需输入群聊名称,不需要包含人数
第4步: 开始执行
  • 点击"开始批量处理"按钮
  • 应用会自动最小化并打开企业微信
  • 自动执行以下操作:
- ✅ 批量通过所有好友申请 - ✅ 记录所有新客户名称 - ✅ 批量邀请到指定群聊

第5步: 查看结果

  • 执行完成后,应用会自动返回
  • 显示操作结果统计:
- 成功通过: X人 - 成功邀请: X人 - 失败: X人

注意事项

  • ⚠️ 执行过程中请勿操作手机
  • ⚠️ 确保手机屏幕保持亮屏
  • ⚠️ 确保网络连接稳定
  • ⚠️ 如果中途失败,可以重新执行
---

功能二: 批量发送消息

准备工作

1. 准备素材库聊天 - 创建一个专门的聊天(可以是个人聊天或群聊) - 在这个聊天中准备好要发送的消息 - 记住这个聊天的名称

2. 准备目标群聊/联系人列表 - 确认要发送消息的群聊/联系人名称 - 确保名称准确无误

操作步骤

第1步: 配置素材库聊天

  • 打开应用,点击"功能二"标签
  • 点击右上角"设置"图标
  • 输入素材库聊天名称
  • 点击"保存"
第2步: 创建消息组
  • 点击"消息组管理"
  • 点击"+"创建新消息组
  • 输入消息组名称(例如: 每日推送)
  • 输入描述信息
  • 设置转发消息数量(例如: 7条)
  • 设置延迟时间范围(例如: 2-5秒)
  • 点击"保存"
第3步: 添加目标群聊
  • 进入刚创建的消息组
  • 点击"添加群聊"
  • 输入群聊/联系人名称
  • 点击"添加"
  • 重复此步骤添加所有目标
第4步: 开始发送
  • 确保企业微信已打开
  • 确保素材库聊天中有足够的消息
  • 点击"开始发送"按钮
  • 应用会自动执行:
- ✅ 打开素材库聊天 - ✅ 选择最新的N条消息 - ✅ 转发到所有目标群聊 - ✅ 自动添加随机延迟

第5步: 查看结果

  • 执行完成后显示统计:
- 成功发送: X个群聊 - 失败: X个群聊 - 失败列表: [群聊名称]

高级功能

智能滚动查找

  • 如果目标联系人不在当前页面
  • 应用会自动向下滚动查找
  • 直到找到或滚动到底部
全角/半角智能匹配
  • 自动处理字符编码差异
  • 例如: 天天一泉~小石榴 和 天天一泉～小石榴 都能匹配
随机延迟
  • 模拟真人操作
  • 避免被检测为机器人
  • 可自定义延迟范围

❓ 常见问题

Q1: 安装时提示"未知来源"怎么办?

A: 需要允许安装未知来源应用 


设置 → 安全 → 未知来源 → 允许
或
设置 → 应用管理 → 特殊权限 → 安装未知应用 → 文件管理器 → 允许

Q2: 无障碍服务开启后自动关闭?

A: 这是部分手机的电池优化导致的 


解决方法:
1. 关闭应用的电池优化
2. 加入自启动白名单
3. 允许后台运行

Q3: 执行过程中卡住不动?

A: 可能的原因和解决方法 


原因1: 网络不稳定
解决: 检查网络连接,切换到稳定的WiFi
原因2: 企业微信版本不兼容
解决: 更新企业微信到最新版本
原因3: 无障碍服务被关闭
解决: 重新开启无障碍服务
原因4: 手机性能不足
解决: 关闭其他应用,释放内存

Q4: 找不到目标群聊/联系人?

A: 检查以下几点 


1. 名称是否完全一致(包括空格、标点符号)
2. 群聊是否在消息列表中(需要有聊天记录)
3. 尝试手动滚动到群聊位置,确认名称
4. 应用会自动滚动查找,请耐心等待

Q5: 部分消息发送失败?

A: 可能的原因 


原因1: 目标群聊不存在或已解散
解决: 检查群聊是否存在
原因2: 没有发送权限(被禁言)
解决: 检查群聊权限
原因3: 网络超时
解决: 增加延迟时间,重新发送

Q6: 会被企业微信检测封号吗?

A: 风险较低,但需要注意 


建议:
1. 设置合理的延迟时间(2-5秒)
2. 避免频繁操作
3. 不要短时间大量发送
4. 模拟真人操作习惯

Q7: 耗电量大吗?

A: 正常情况下耗电很少 



  • 应用只在执行任务时运行
    • 

  • 执行完成后自动停止
    • 

  • 不会常驻后台
    • 

  • 几乎不影响续航
    •

Q8: 可以在后台运行吗?

A: 可以,但需要配置 


1. 关闭电池优化
2. 允许后台运行
3. 加入自启动白名单
4. 执行时保持屏幕亮屏(推荐)

Q9: 支持多个企业微信账号吗?

A: 目前只支持当前登录的账号 


如需切换账号:
1. 退出当前企业微信账号
2. 登录新账号
3. 重新执行操作

Q10: 如何更新应用?

A: 获取新版APK重新安装 


1. 下载最新版APK
2. 直接安装(会覆盖旧版)
3. 数据会保留
4. 无需卸载旧版

⚠️ 注意事项

使用前必读

1. 确保无障碍服务已开启 - 这是应用的核心权限 - 未开启将无法使用

2. 执行时请勿操作手机 - 避免干扰自动化流程 - 可能导致执行失败

3. 保持屏幕亮屏 - 部分手机锁屏后无障碍服务会暂停 - 建议执行时保持屏幕常亮

4. 确保网络稳定 - 网络不稳定可能导致操作失败 - 推荐使用WiFi

安全建议

1. 合理使用自动化功能 - 不要过度频繁使用 - 避免被企业微信检测

2. 设置合理的延迟时间 - 模拟真人操作 - 建议2-5秒随机延迟

3. 定期检查执行结果 - 确认消息是否发送成功 - 及时处理失败的情况

4. 备份重要数据 - 定期导出消息组配置 - 避免数据丢失
